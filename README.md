# X-penses

The application helps you to track all of your expenses with a user friendly UI.It is built with ` Next.js,`  `Node.js`, `Express.js`, and `JWT`, empowers you to manage your finances effectively.  
It offers features for daily and monthly expense tracking, insightful visualizations, and secure user authentication.

### Tools and Tech Stack:

#### Front-end 
<ul>
  <li>TypeScript</li>
  <li>TailwindCSS</li>
  <li>NEXT js</li>
  <li>Chart js</li>
</ul>

#### Back-end
<ul>
  <li>TypeScript</li>
  <li>NODE js (Env) </li>
  <li>Express js</li>
</ul>

#### Database
<ul>
  <li> PostgreSQL </li>
</ul>

**Libraries :** 
`nodemailer` , `jsonwebtoken(JWT)` `Joi` `Sequelize-typescript` `pg`  


# Featues
<ul>
  <li> Choose between Dark and Light theme. </li>
  <li>Authentication Authorization , (User can login , logout, Token based authentication) </li>
  <li> Forget Password using OTP verification mail. </li>
  <li>Add , Edit, Delete Expenses </li>
  <li>Add expenses with the appropriate category </li>
  <li>Filter your expenses based on the categories </li>
  <li> Analytics , Dashboard to view daily expenses graphically  </li>
  <li> Piechart to show the category based expenses , Barchart for analyzing daily expenses </li>
</ul>
